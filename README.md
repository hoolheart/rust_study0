## 2021-12-14

第一个例子，猜数字：

1. 使用`rand`获取一个100以内的随机自然数；
2. 要求用户输入猜测值；
3. 反馈太小或太大的提示，直至用户猜到之前产生的随机数，然后退出；
4. 若输入不符合要求，打印提示并重试。
